# IMT542 Information Structures Access Assignment

This project demonstrates three different information structures with three different access technologies.

## Information Structures and Access Methods

1. JSON  
   - Access technology: API connection over HTTP  
   - Source: openFDA Food Enforcement API  

2. HTML  
   - Access technology: Python code library reading HTML over HTTP  
   - Source: Python official website  

3. CSV  
   - Access technology: Local file creation and local file reading  
   - Source: Sample CSV file created by the Python script  

## Environment

This project is designed to run in a standard Python 3 environment.

Recommended environment:

- Python 3.9 or newer
- Internet access
- pip enabled

This project can be run in:

- VS Code
- Terminal / Command Prompt
- Google Colab
- Jupyter Notebook
